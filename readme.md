# JSE
Java Standard Edition
## Import a repository from Eclipse
1. Go to File / Import
2. Select Git / Projects from Git
3. Select Clone URI
4. Enter the location of the Git repository:
- URI: https://github.com/llabor/JSE/
- Host: github.com
- Repository path: /llabor/JSE/
